# To-Do List CLI App

A simple command-line To-Do application that allows you to add tasks and list them with their status.

## Features
- Add tasks with a title and optional description  
- List all tasks with their status (done / not done)  
- Each task has an auto-generated ID  

## How to Run
1. Make sure Python 3 is installed.  
2. Run the script using:

   python3 to_do.py
